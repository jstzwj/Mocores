#简介
Mocores是一个利用闲置的计算机资源进行分布式计算的软件系统。与其他分布式系统不同，该系统的子节点可以是不可靠的，网络通信不佳的。该系统适用于并行粒度较大的问题的解决。

#兼容性
这个软件使用c++11编写。它在MSVC15和GNUC4.9.2编译器中测试。
